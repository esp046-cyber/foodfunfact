# foodfunfact
Features:
Beautiful gradient design with purple/pink color scheme
Animated food emoji that bounces and changes with each fact
20 interesting food facts that appear randomly
Click counter to track how many facts you've discovered
Smooth animations and hover effects
Fully responsive design that works on all devices
Just click the "Get Fun Fact!" button to learn fascinating trivia about food. Each fact comes with a relevant emoji that animates at the top. Enjoy exploring these fun food facts!
